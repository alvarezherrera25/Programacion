La aritmética modular es un sistema aritmético para números 
enteros, donde los números "dan la vuelta" al alcanzar un 
cierto valor, denominado módulo, similar a cómo funcionan 
las horas en un reloj.
EJ: Es conocida comúnmente como "aritmética del reloj", 
ya que si el módulo es 12, después de las 12 viene la 1.

                    insertar (Jose Mari)
                    
                    aux = lista;
                    lista.dato = (el priemro de lista) Adolfo
                    aux.dato = Adolfo
                    aux.enlace.dato= (el segundo de la lista) Leopoldo

                    while (aux enlace != null){
                        aux = aux.enlace;
                    }
                    aux.enlace = nuevo; 


                    1. nuevo = new Nodo(dato);
                    Creas el objeto en memoria (el nuevo vagón).
                    
                    2. nuevo.enlace = lista;
                    El nuevo nodo "mira" hacia donde apuntaba la lista originalmente. Así no pierdes los elementos que ya tenías.
                    
                    3. lista = nuevo;
                    Ahora la variable lista (la cabeza) apunta al nuevo nodo. A partir de este momento, ese objeto es oficialmente el primero.
                    ej: 
                    public void agregarAlInicio(int dato) {
                        Nodo nuevo = new Nodo(dato); // Crear el objeto
                        nuevo.enlace = lista;        // Conectar con lo anterior
                        lista = nuevo;               // Actualizar el inicio
                    }
